## このリポジトリで管理するもの

- **スイート横断イシュー** — 共通OAuth、ポータルサイト、共通ナビゲーション、ブランドガイドライン
- **次期プロダクト検討** — 新プロダクト候補のリサーチ・仕様書
- **マーケティング戦略** — PLG設計、SEO戦略、ユーザー獲得施策
- **テクニカルアーキテクチャ** — 認証統一、データ共有、インフラ設計

## このリポジトリで管理しないもの

- 各プロダクト固有のバグ・機能追加 → 各リポジトリのIssueへ
- コード → 各リポジトリへ
