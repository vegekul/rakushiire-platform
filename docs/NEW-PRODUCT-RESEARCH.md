## TOP 3 詳細

### 優先度 S-1: 飲食店原価率チェッカー

**Issue**: [rakushiire-platform #6](https://github.com/vegekul/rakushiire-platform/issues/6)

#### なぜ今やるべきか

- 飲食店の **94.6%が仕入れ価格上昇に直面**（2025年帝国データバンク調査）
- 既存競合（Recipro、HANJO Kitchen）は月額1〜3万円+初期費用20万円以上で**個人店向けが空白**
- ラクシーレはすでに **3,567品の仕入れ価格DB** を保有 → ほぼゼロコストで実現可能
- 仕入れチェッカーとの相互補完: 「食べログURL→食材解析→レシピ原価率試算→代替卸提案」が1本のストーリーに

#### 機能概要

- レシピを入力（食材名・使用量・単位）
- ラクシーレ卸業者DBの実価格で原価率を自動計算
- FL比率（食材費+人件費 ≦ 売上60%）の視覚化
- 「この食材を安い仕入れ先に変えると原価率が○%下がる」提案

#### PLG設計

| ティア | 内容 | 価格 |
|---|---|---|
| 無料 | レシピ3件まで試算 | ¥0 |
| スタンダード | 無制限保存 + 価格変動アラート + CSVエクスポート | 月額¥980〜 |

#### SEOキーワード

「飲食店 原価率 計算」「飲食店 原価率 下げる」「レシピ 原価計算 飲食店」

#### 技術要件

- 既存転用: 卸業者DB・matchingエンジン・Gemini食材解析
- 新規開発: レシピ登録モデル（Prisma）+ 計算UI
- **開発期間目安**: 2〜3週間（MVP）

### 優先度 S-2: 飲食店食材価格アラート

**Issue**: [rakushiire-platform #7](https://github.com/vegekul/rakushiire-platform/issues/7)

#### なぜ今やるべきか

- `ai-procurement-optimizer` の ROADMAP Phase 4 で以下が**すでに実装済み**:
  - PriceHistory テーブル + 日次 Cron スナップショット
  - 10%以上の価格変動検知アラートロジック
  - Nodemailer / LINE Messaging API 通知基盤
- **UIと登録フローを整えるだけで独立プロダクトとして公開可能**
- 競合は実質ゼロ（農水省の価格調査は月次で遅すぎる）

#### 機能概要

- 食材名入力 → メール/LINE で価格変動を通知
- 変動幅しきい値設定（5% / 10% / 20%）
- 週次レポート（価格推移チャート）
- 「今すぐ安い仕入れ先に切り替える」→ 仕入れチェッカーへ

#### PLG設計

| ティア | 内容 | 価格 |
|---|---|---|
| 無料 | 食材3品まで監視、メール通知のみ | ¥0 |
| スタンダード | 監視無制限 + LINE通知 + 週次レポート | 月額¥490〜 |

#### SEOキーワード

「飲食店 食材 価格 高騰」「野菜 仕入れ値 変動 通知」「食材 価格 アラート」

#### 技術要件

- **新規開発最小**: 監視登録UI + しきい値設定フォーム + 通知テンプレート
- **開発期間目安**: 1週間（MVP）

### 優先度 A-1: 飲食店口コミ分析・返信AI

**Issue**: [rakushiire-platform #8](https://github.com/vegekul/rakushiire-platform/issues/8)

#### なぜ今やるべきか

- 食べログURLを起点とする仕入れチェッカーユーザーと**完全に重複するターゲット**
- 既存の `src/lib/gemini/` と `src/lib/tabelog/` を転用してほぼ実装可能
- 「口コミで指摘された品質問題→使用食材・仕入れ先を見直す→代替卸提案」という**ラクシーレ固有の体験**

#### 機能概要

- 食べログURL or 口コミテキストを入力
- AIが感情分析（ポジティブ / ネガティブ / 改善要望を分類）
- 種別ごとに返信テンプレートを生成（丁寧 / カジュアル / お詫び）
- 「この食材品質を改善するには」→ 仕入れチェッカーへ誘導

#### PLG設計

| ティア | 内容 | 価格 |
|---|---|---|
| 無料 | 月10件まで分析 + 返信案生成 | ¥0 |
| スタンダード | 無制限 + 競合比較 + 自動投稿スケジューリング | 月額¥1,980〜 |

#### SEOキーワード

「食べログ 返信 書き方」「Googleマップ 口コミ 返信 飲食店」「飲食店 口コミ 管理」
