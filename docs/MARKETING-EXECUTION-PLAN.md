## 0. 戦略の一言要約

> **「食材価格が上がるたびに、ラクシーレが日本中の飲食店のスマホに出てくる状態を作る」**

食材高騰は毎月話題になる。そのたびにラクシーレが検索上位に出て、SNSで流れて、メディアで取り上げられる。
「仕入れのことならラクシーレ」というポジションを12ヶ月かけて確立する。

| 責任範囲 | 主要KPI |
|---|---|
| GA4 / Search Console 設定・分析 | セッション数・直帰率・CVR |

| ファネル分析（どこで離脱しているか） | ステップ別転換率 |
| メール配信設定（招待・ナーチャリング） | 開封率（目標：30%以上）・クリック率（目標：8%以上） |
| ヒートマップ分析（ユーザー行動観察） | スクロール率・クリック分布 |

**週次アクション:**
- 月曜: 先週のGA4数値をSlackに共有（セッション・CVR・検索流入TOP10）
- 水曜: LPのA/Bテスト結果確認・改善を実装
- 金曜: 翌週の施策優先順位をCMOに提案

**重点施策（4〜6月）:**

1. **登録フローの徹底最適化**
   - 現状: LPから登録まで何ステップかを計測
   - 目標: 登録完了率 60%以上（途中離脱を50%削減）
   - 施策: プログレスバー表示・入力項目を最小化・エラーメッセージの改善

2. **Aha Moment の短縮**
   - 現状: 登録 → 初回レポート生成まで何分かかるか計測
   - 目標: 登録から3分以内にレポートを見せる
   - 施策: 食べログURL入力をLP上でそのまま始められる設計

3. **メールナーチャリングシーケンス**
   ```
   登録直後:  「レポートの見方ガイド」（開封率最重要）
   3日後:     「納品書アップロードで月次コストを追跡しよう」
   7日後:     「同業態の事例: 居酒屋オーナーの仕入れ改善事例」
   14日後:    「食材価格が上がっている今、仕入れ先を比較してみましょう」
   30日後:    「1ヶ月間ありがとうございます。ご感想をお聞かせください」
   ```

**コピーライティング原則（ラクシーレブランドの声）:**

```
【トーン&マナー】
- 難しいことを、わかりやすく
- データで語る（「安くなる」より「年間120万円削減の可能性」）
- 正直さを武器にする（「全店が削減できるわけではありません」）
- 飲食店オーナーに寄り添う（上から目線NG）

【NGワード】
- 「革命的」「圧倒的」「業界初」（証拠なし）
- 「簡単」（手間をゼロにするわけではない）
- 「必ず」「絶対」（保証できない）

【OKワード】
- 「3分でわかる」（具体的）
- 「年間○万円の削減可能性を発見」（数値）
- 「食べログURLを貼るだけ」（動作が明確）
- 「今の仕入れ先が正解かどうか確認する」（不安に寄り添う）
```

**キャッチコピー候補（LP・広告・PR用）:**

```
メインキャッチ（最有力）:
「食べログURLを貼るだけ。年間○○万円の削減可能性が3分でわかる。」

バリエーション①（緊急性）:
「食材が値上がりしているのに、仕入れ先はいつから変えていませんか？」

バリエーション②（実績訴求）:
「ベジクル取引先2,500店が導入。AI仕入れチェッカー、無料でお試し。」

バリエーション③（共感）:
「『相見積もりを取りたいけど面倒』を解決しました。」

SNS用（短く・バズ狙い）:
「飲食店オーナーさんへ。今の仕入れ、本当に最安値ですか？」
「キャベツ1玉〇円、うちで仕入れてたら月△万円の損でした（スクショ）」
```

## 3. バズ施策 特集

### バズを狙う「仕掛け」一覧

#### 仕掛け①: 食材価格高騰ニュース即応ツイート

```
トリガー: 農水省・日経新聞・NHKが「○○の価格が高騰」報道
反応時間: 報道から2時間以内

投稿例:
「【速報】愛知産キャベツが異常高騰
産地価格: ○円/kg（平年比+48%）

ラクシーレの食材相場チェッカーで確認したところ、
東京市場への影響は○〜○円/kgの見込み。

代替食材として春菊・ほうれん草への切替が有効。
→ 詳細データはこちら: [リンク]

#飲食店 #食材高騰 #仕入れ #キャベツ価格」

狙い: 飲食店オーナー・食品業界関係者・農業メディアのRTを誘発
```

#### 仕掛け②: レポートのOGP動的生成でユーザーが自然に拡散

```
設計:
「シェアする」ボタンを押したとき、このカードが表示される:

┌─────────────────────────────────┐
│ 🥬 飲食店食材仕入れチェッカー       │
│                                 │
│ 【○○レストラン 診断結果】         │
│ 年間削減可能額: 87万円            │
│ 最も割高な食材: 豚バラ肉          │
│                                 │
│ あなたの店舗も無料診断 →          │
│ rakushiire.jp                   │
└─────────────────────────────────┘

効果: ユーザーが「こんなに削減できるの！？」と自慢したくなる
→ ユーザーが自発的にSNS拡散する仕組み
```

#### 仕掛け③: 「正直すぎる事例記事」でメディア掲載を狙う

```
タイトル例:
「焼肉店の牛肉仕入れ、変えたくても変えられなかった話【ラクシーレ事例】」

本文で正直に書く:
- 診断の結果、年間40万円の削減余地が見つかった
- しかしオーナーは「今の業者との10年の付き合いがある」と仕入れ先を変えなかった
- それでも「削減余地を知れた価値はあった」と話す

なぜバズるか:
- 「失敗事例」「変えられなかった話」は飲食業界の本音と共鳴する
- メディアは「成功だけを語らない誠実さ」を好む
- 飲食店コミュニティでRT・保存されやすい
```

#### 仕掛け④: 「食材費率 業態別ランキング」年次レポートのバズ

```
毎年12月に発表するデータレポート:
「2026年 飲食店食材費率ランキング（ラクシーレ独自データ）」

内容:
- 業態別 平均食材費率（定食・居酒屋・焼肉・中華・ラーメン等）
- 今年最も値上がりした食材TOP10
- 地域別 食材コスト比較（東京 vs 大阪 vs 地方）
- 「最もコスト改善に積極的だった業態」

プレスリリースで配信 → 日経MJ・食品業界紙が記事化
Xでインフォグラフィックを投稿 → 飲食店コミュニティでバズる
```
